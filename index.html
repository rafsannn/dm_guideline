The exact moves that convert preparation into marks. Study this the night before.

Q
The Quick Win Blueprint
3 guaranteed mark sources
01
Truth Tables (~5.5 marks) — Mechanical, predictable. Master the implication arrow $p \rightarrow q$. The compound form $(p \rightarrow q) \vee (\neg p \rightarrow q)$ is a recurring favourite. Never miss these marks.
02
Double Summation — Evaluate the inner loop first, then outer. It resolves to simple arithmetic. Also write it as a C for loop — that's easy bonus marks on its own.
03
Definitions — "Define Power Set," "What is a Tautology?" — memorize these verbatim from the slides. Free marks if you've read them once.
T
Critical Logic Translations
English → symbolic form
"p only if q" → $p \rightarrow q$
"p is necessary for q" → $q \rightarrow p$
"p is sufficient for q" → $p \rightarrow q$
"neither p nor q" → $\neg p \wedge \neg q$
"p unless q" → $\neg q \rightarrow p$
"whenever p, q" → $p \rightarrow q$
🎯
Choose Groups Wisely
Answer 2 of 3 groups. Group A (Ch 1+2) gives the broadest coverage. Only tackle Group B (Ch 3) if you've fully drilled counting theory.
🔺
The Big Three Proofs
Be ready to prove: (1) logical equivalences via truth tables, (2) $A \times B \neq B \times A$ using concrete sets, (3) function injectivity / surjectivity via diagrams.
⚡
English Fluency = Marks
The teacher uses "whenever," "necessary for," "unless," and "sufficient" to test translation. Drill these five phrases until the conversion is instinctive.
∑
Double Sum = Free Marks
Predictable pattern, high marks. Evaluate inner summation, substitute, then evaluate outer. Also write the equivalent nested C for loop — two answers in one question.
📐
Set Proofs Structure
For $A \times B \neq B \times A$: define $A=\{a,b\}$, $B=\{1,2\}$, show $(a,1) \in A\times B$ but $(a,1) \notin B\times A$. One counterexample is all you need.
📖
Beyond the Slides
Pigeonhole Principle, $P(n,r) = C(n,r) \cdot P(r,r)$ proof, and Rules of Inference have appeared. Read these one textbook chapter sections before exam day.
